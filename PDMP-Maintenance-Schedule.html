PDMP Maintenance Schedule

The Maintenance Schedule feature in RxConsole enables State PDMP administrators to create and manage scheduled maintenance events. These events may include activities such as application updates, software installations, or operating system configurations.

During a maintenance window, the PDMP system will be taken offline and will not be able to process incoming requests or generate responses. If a request is received during this time, the system will return an error code "DM", indicating a processing error due to scheduled maintenance. Once the maintenance period concludes, the system will resume normal operations.

HeadingDescription
FromThe date and time for when the maintenance event will begin.
ToThe date and time for when the maintenance event will end.
TypeThe event type for which the maintenance event is scheduled.
MessageAn informative note about the respective event.
StatusThe status of the scheduled event, such as "Completed" or "Cancelled".

16.1 Create and Modify a Maintenance Event

  1. Click on the PDMP Maintenance Schedule button, located on the left-hand side of the screen.
  2. Click on the Add Maintenance button located in the top right-hand corner of the screen.
Note: You can only create a new maintenance event if no other upcoming event is scheduled. If there's a conflict, an alert message will appear.
  1. A pop-up screen titled Maintenance will appear.
  2. Select the start date and time by clicking the blue calendar in the field labeled From Date & Time.
  3. Select the end date and time by clicking the blue calendar in the field labeled To Date & Time.
  4. Select the reason for the maintenance event from the Reason Type dropdown.
  5. Add a note into the Reason Message text box to add context to the maintenance event.
  6. Press the Save button to add the new maintenance event.

To modify an already existing maintenance event:

  1. Click on an event listed in the Maintenance Schedule.
  2. Modify any information in the Maintenance popup.
  3. Press the Save button to record the changes.
  4. Press the Complete button to mark the event as completed.
  5. Press the Cancel button to cancel the event in the maintenance schedule.
