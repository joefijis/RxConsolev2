User Management

State PDMP Administrators are responsible for managing user access within their systems. This includes the ability to add new users with the user type "System User."

In addition, State Administrators can view a list of existing PDMP Administrators and System Users, and they have the authority to edit details for current System Users. To modify information related to an existing PDMP Administrator, the State Administrator must submit a request to the RxCheck Administrator.

14.1 Search for and Update User Information

  1. Click on the User Management button, located on the left-hand side of the screen.
  2. Locate the user you are searching for by:
    • Scrolling through the list of users displayed on the screen, or
    • Typing the user's name into the Filter User search bar in the top right corner.
  3. Select the desired user by clicking on their information to view or modify their record:
    • If the user type is System User, a pop-up screen titled Add/Update User will appear.
    • If the user type is PDMP Admin, a pop-up notification will appear to inform you to reach out to the RxCheck Administrator.
  4. For system users, you may edit any information (except the password) displayed in each data field. Click the Save button to record the changes.
  5. Verify that the user has been successfully updated by looking for the alert: "Selected User Updated Successfully".

14.2 Adding Users

  1. Click on the User Management button, located on the left-hand side of the screen.
  2. Click on the Add User button located in the top right corner of the screen.
  3. In the popup window, enter the requested information into the appropriate data field.
  4. Click on the Save button to record the information for your new user.
  5. Verify that the new user has been successfully added by looking for the alert: "New User Created Successfully".

14.3 Unlocking a User Account

  1. Click on the User Management button, located on the left-hand side of the screen.
  2. A user account that is locked will display the "lock" icon next to the account status.
  3. Click on the lock icon to unlock the account for the corresponding user.
  4. Verify the account has been unlocked by looking for the alert: "User Successfully unlocked."
